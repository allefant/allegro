# Audio codecs

## API: al_init_flac_addon

This function to registers [al_load_sample_flac] with [al_load_sample] to
handle files with the extension ".flac".  You will need to include the
`a5_flac.h` header file and link with the `a5_flac` library.

Return true on success.

## API: al_load_sample_flac

Loads a sample in FLAC format.

See also: [al_load_sample]

## API: al_init_ogg_vorbis_addon

This function to registers [al_load_sample_ogg_vorbis] with [al_load_sample]
and [al_load_stream_ogg_vorbis] with [al_stream_from_file] to handle files
with the extension ".ogg" (assumed to contain Vorbis data).  You will need to
include the `a5_vorbis.h` header file and link with the `a5_vorbis` library.

Return true on success.

## API: al_load_sample_ogg_vorbis

Loads a sample in Ogg Vorbis format.

See also: [al_load_sample]

## API: al_load_stream_ogg_vorbis

Loads a stream in Ogg Vorbis format.

See also: [al_stream_from_file]
